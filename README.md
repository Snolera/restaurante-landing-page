# Leading Page de um Restaurante Fictício

## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma *Leading Page* para um restaurante fictício, utilizando **HTML, CSS e JavaScript**.

O principal objetivo deste projeto é praticar e aprimorar meus conhecimentos no desenvolvimento de sites utilizando essas tecnologias. Durante o processo, percebi a importância de melhorar alguns pontos, como:

- Uso de **HTML semântico**
- Melhor **organização e estruturação do código**
- Abstração do projeto com base nos princípios de **Clean Code**
- Separação mais clara de responsabilidades

Um dos maiores desafios enfrentados foi o **posicionamento das divs e organização do layout com CSS**. No entanto, após prática e ajustes, hoje já compreendo melhor conceitos como:

- `display: flex`
- `position`
- Organização estrutural de containers
- Hierarquia de elementos

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, aprendi que:

- Planejamento da estrutura antes de codar facilita o processo
- Organização do CSS impacta diretamente na manutenção
- Pequenos ajustes na estrutura HTML fazem grande diferença na responsividade
- A prática constante melhora a compreensão de posicionamento e layout
